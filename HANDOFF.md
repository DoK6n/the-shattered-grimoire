## 4) 작업 방식

- **원고 작성**: `chapters/chXX.md`에 마크다운으로 작성. 한 파일 = 한 화.
- **포맷 규칙**:
  - `## 소제목` — 섹션 헤더 (번호 X)
  - `**굵게**` — 효과음, 결정적 순간, 강조
  - `*기울임*` — 속마음, 내면 독백
  - `>> 텍스트` — 마도서 원문 (인용 스타일)
  - `> 텍스트` — 일반 인용
  - `---` 절대 사용 금지
- **빌드**:
  ```
  python C:\Users\ehrbs\projects\dok6n\build_docs.py
  ```
  결과물: `조각난_마도서/03_본문/XX화_제목.docx`
- **새 화 추가 시**: `build_docs.py` 하단 `build_doc(...)` 호출 블록에 한 줄 추가.
  예:
  ```python
  build_doc('03_본문/17화_제목.docx', '17화 — 제목', _chap(17))
  ```
  그리고 상단의 `N1, N2, ...` 언패킹 범위도 `range(1, 18)` 등으로 늘릴 것.
