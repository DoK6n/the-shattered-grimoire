# 새 화 작업 부트스트랩

다음을 순서대로 읽고 ch$ARGUMENTS 작업 준비를 완료하세요. 대화 맥락이 없는 새 세션에서도 이 한 명령으로 모든 컨텍스트가 복원되어야 합니다.

## 1. 메타·세계관 컨텍스트 로드
1. `CLAUDE.md` — 프로젝트 지침 (집필 규칙, 매너리즘 가이드, 에이전트 호출 정책)
2. `HANDOFF.md` — 직전 마감 화 / 누적 상태 / 진행 중인 떡밥 / 매너리즘 카운트 / 직전 5화 검수 권고 패턴

## 2. 직전 화 본문 로드
3. `chapters/ch{N-2:02d}.md` — 두 화 전 본문 (씬 연결 컨텍스트)
4. `chapters/ch{N-1:02d}.md` — 직전 화 본문 (직접 이어지는 씬)

## 3. 회차 구성 확인
5. `조각난_마도서/02_회차구성/100화_아크구성.docx` — ch$ARGUMENTS 항목의 핵심 사건 / 부 위치 / 다음 화와의 관계

## 4. 검수 로그 (있다면)
6. `chapters/_review_logs/ch{N-1:02d}.md` — 직전 화 검수 통합 로그 (있을 때만)

## 5. 보고
모든 자료를 읽은 뒤, 다음 형식으로 한 페이지 보고:

```
# ch$ARGUMENTS 작업 준비 완료

## 핵심 사건 (아크구성표)
- ...

## 직전 마감 (ch{N-1}) 결말 한 단락
> "..."

## 누적 상태 (HANDOFF.md 기준)
- 시간선: ...
- 인물 위치: ...
- 마나 풀: ...% / 페이지 N장 / 부상 누적: ...
- 매트릭스: ...
- 진행 중인 떡밥: ...

## 매너리즘 가이드 (의식할 자리)
- 한 결: 명사 의미만
- 한 호흡: 영창 안만
- 자기: 3인칭 금지
- 자리 잡았다: 분산
- chN 본문 노출 금지 — 작중 시간선으로 변환

## 직전 5화 검수 권고 패턴
- ...

## 집필 시작 가능 여부
- 준비 완료 / 추가 확인 필요(...)
```

## 6. 집필 진행
보고 직후, 사용자 승인을 기다리지 말고 바로 **5000~5500자 집필**을 시작하세요. 화 마감 후엔 `review-team` 라운드로 검수합니다.

## 7. 마감 후 자동 트리거
화 통과 후엔 자동으로 `novel-handoff` 스킬을 호출하여 `HANDOFF.md`를 갱신합니다.
